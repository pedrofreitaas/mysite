  • Language used: Python.

Password Manager

  • Simple password manager that, based on a master key, handles and storages passwords for a user.
  • Uses encryption based in the Fernet python's module.
  • Uses a hidden shelve to storage the passwords encrypted versions.
  • Can create a password randomly, based in the users configurations.
  • Suports: input/creation/deletion/list of passwords, and change of configurations, all based in a master key.
  • Language used: Python
